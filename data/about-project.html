مشروع (شوية كيمياء)

عبارة عن منصة افتراضية تحوي معلومات عن منهج الكيمياء بشكل مبسط.
يعتبر هذا التطبيق مشروعًا لمادة كيمياء 4.
معلم المادة: شيخان الجفري.
 تصميم وبرمجة الطالب: حمد محمد جمال الدين.
 المشروع عبارة عن موقع ويندرج تحت مسمى Web Application (تطبيق ويب) أي تطبيق متعدد المنصات يعمل على المتصفح... على أي منصة سواءً كانت جهاز حاسوب أو لاب توب محمول وحتى الهواتف الذكية.

برمجت هذا المشروع باستخدام لغات الويب والتقنيات الحديثة:-

لغات البرمجة المستخدمة (Front-End):

  • HTML
  • CSS
  • JavaScript
إطارات العمل المستخدمة (Framworks):
  • VueJS 2 (CDN)
  • BootStrap 5.0 (CDN)
